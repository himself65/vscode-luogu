# vscode-luogu

轻松的让你在 VSCode 上提交洛谷代码

## 使用说明

#### **查看题目**

1. 按下 `F1` 或 `Ctrl+Shift+P` ，查找`Luogu: 查看题目`命令，点击后输入题号完成搜索

2. 点击界面左侧洛谷图片，然后点击搜索图标，输入题号完成搜索

## 完成功能

- 查看题目


## 开发中功能

- 登录账号

- 提交代码

- 题目离线查看

- 查看自己测评

- 查看、发布犇犇
